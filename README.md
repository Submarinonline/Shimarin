# Shimarin(仮)
[Submarin](https://submarin.online) を表示
## 操作方法
- Control+Q...Quit
- Control+R or F5...reload
- F11...全画面のやつ
- F12...DevToolsを開く
## 使い方
[releases](https://github.com/shaaaaaQ/Shimarin/releases)から最新のやつをダウンロード  
Linux
```
./Submarin-Electron
```
Windows
```
Submarin-Electron.exe
```
